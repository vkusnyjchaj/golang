# Указатели

В Go существуют указатели (pointers). Указатели содержат адрес памяти по которому находится значение.

`*T` указатель на значение `T`. Нулевым значением указателя является `nil`.

`var p *int`

Оператор `&` генерирует указатель для переменной:
```
i := 42
p = &i
```

Чтение значения через указатель:
`fmt.Println(*p)`

Запись значения через указатель:
`*p = 21`

Чтение и запись через указатель известны как *разыменование* или *косвенное обращение*. 

Разыменование - это операция, позволяющая получить доступ к значению, хранящемуся по адресу, который находится в переменной типа указателя.
Разыменование используется для передачи значения в функцию по ссылке. Это позволяет не только изменять исходную переменную вне функции, но и экономить память
при передачи больших значений (передавать только адрес, а не копировать само значение).

В отличие от С в Go нет арифметики для указателей.
