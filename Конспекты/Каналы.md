# Каналы

Каналы в Go представляют собой способ коммуникации между горутинами.

Канал можно создать с помощью `make`:
```
ch := make(chan int)
```

Получение данных из каналов выглядит следующим образом:
```
ch <- v    // Send v to channel ch.
v := <-ch  // Receive from ch, and
           // assign value to v.
```

В Go канал chan int может быть буферизированным - в таком случае он хранит некоторое количество значений до того, как чтение их вынесет из канала.

Размер буфера указывается при создании:
```
ch := make(chan int, 100)
```
