# Каналы

Каналы в Go представляют собой способ коммуникации между горутинами.

Канал можно создать с помощью `make`:
```
ch := make(chan int)
```

Получение данных из каналов выглядит следующим образом:
```
ch <- v    // Send v to channel ch.
v := <-ch  // Receive from ch, and
           // assign value to v.
```

В Go канал chan int может быть буферизированным - в таком случае он хранит некоторое количество значений до того, как чтение их вынесет из канала.

Размер буфера указывается при создании:
```
ch := make(chan int, 100)
```

Можно проверить был ли закрыт канал с помощью второго аргумента при передаче из канала:
```
v, ok := <-ch
```

ok = false => канал закрыт

Обычно каналы не нужно закрывать, это используется только тогда когда нужно сообщить получателю что больше значения не поступают.

*Важно* 
Закрытие канала сигнализирует всем читателям о том, что больше данных не будет. После этого любые дальнейшие операции записи в канал станут паникой (panic)
