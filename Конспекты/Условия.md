# Условия

Условия в Go подобно циклам не требуют круглых скобок, но требуют фигурных вокруг тела условия:
```
if x < 0 {
  return sqrt(-x) + "i"
}
```

В условие можно поместить объявление переменной. Переменная будет доступна только внутри тела условия:
```
if v := math.Pow(x, n); v < lim {
  return v
}
```

Переменные объявленные в блоке `if` также доступны во всех блоках `else`:
```
if v := math.Pow(x, n); v < lim {
  return v
} else {
  fmt.Printf("%g >= %g\n", v, lim)
}
// can't use v here, though
```
