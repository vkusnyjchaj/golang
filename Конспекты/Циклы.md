# Циклы

В Go существует толька одна конструкция для циклов - `for`. 

Цикл `for` включает в себя три компонента:
- инициализатор, выполняется перед первой итерацией
- условное выражение, выполняется перед каждой итерацией
- пост-итерация, выполняется в конце каждой итерации

Инициализатор часто может представлять собой короткое объявление переменной, область видимости которой ограничивается областью тела цикла.

Цикл перестанет выполняться как только условное выражение вернет `false`.

В отличие от других языков программирования в Go не нужно указывать круглые скобки вокруг компонентов цикла и **обязательно** нужно указывать фигурные скобки вокруг тела цикла.

Если инициализатор и пост-итерация в цикле `for` не заданы, то можно опустить знаки `;` и цикл превратится аналог `while` в других языках:
```
for sum < 1000 {
  sum += sum
}
```

Если опустить и условие, то цикл `for` превратится в бесконечный цикл:
```
for {
}
```
