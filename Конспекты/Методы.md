# Методы

В Go нет классов, однако можно добавлять методы к типам:

Метод в Go это функция со специальным аргументом получателеми (`receiver`).

Получатель указывается в собственном списке аргументов между ключевым словом `func` и имененем метода.

В примере ниже получатель имеет тип `Vertex` и имя `v`:
```
type Vertex struct {
	X, Y float64
}

func (v Vertex) Abs() float64 {
	return math.Sqrt(v.X*v.X + v.Y*v.Y)
}

func main() {
	v := Vertex{3, 4}
	fmt.Println(v.Abs())
}

```

Метод - это просто функция с аргументом получателем.
