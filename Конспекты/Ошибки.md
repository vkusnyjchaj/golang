# Ошибки

Тип ошибок в Go это встроенный интерфейс:
```
type error interface {
    Error() string
}
```

Функции часто возвращают ошибку и вызывающий код должен обработать ошибки тестируя равняется ли значение ошибки nil:
```
i, err := strconv.Atoi("42")
if err != nil {
    fmt.Printf("couldn't convert number: %v\n", err)
    return
}
fmt.Println("Converted integer:", i)
```
