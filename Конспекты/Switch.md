# Switch

В Go `switch` устроен аналогично большинству других языков, за исключением того что Go выполняет только выбранный вариант `case`, а не все - `break`, необходимый для остановки в других языках, в Go добавляется автоматически.

Значения `case` могут быть любыми выражениями, которые вычисляются во время выполнения (runtime), а не только константами как в других языках.

И тип значения в `switch` может быть любым, не только целым числом — строка, булево значение, результат функции и т.д.

Конструкция `switch` без условия аналогична `switch true` и может быть использована для написание длинных цепочек if-then-else.

Как и в других языках для `switch` есть вариант `default`.
