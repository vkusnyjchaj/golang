# Panic

`panic` - встроенная функция, которая останавливает обычное выполнение и начинает процесс паники.

Когда функция F вызывает panic, выполнение F прекращается, все отложенные функции в ней выполняются, и затем F возвращается вызывающей функции.

Для вызывающей стороны F ведёт себя так, будто сама вызвала panic. Процесс продолжается вверх по стеку вызовов, пока все функции в текущей горутине не завершатся - после этого программа аварийно завершается.

Паника может быть вызвана напрямую с помощью `panic`, а может возникнуть из-за ошибок времени выполнения, например, выхода за границы массива.
