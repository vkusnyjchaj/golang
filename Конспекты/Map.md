# Map

Карта представляет собой коллекцию из ключей и связанных с ними значений. Нулевое значение `nil`.

В непроинициализированную карту с нулевым значением нельзя добавлять пары ключ-значение.

Карту можно создать и проинициализировать через функцию `make`:
```
m = make(map[string]Vertex)
	m["Bell Labs"] = Vertex{
		40.68433, -74.39967,
	}
```
где ключи будут иметь тип string, а значения тип Vertex (структура c lat/lon).

Карты можно объявлять через литералы подобно структурам, отличие в том, что ключи обязательны:
```
var m = map[string]Vertex{
	"Bell Labs": Vertex{
		40.68433, -74.39967,
	},
	"Google": Vertex{
		37.42202, -122.08408,
	},
}
```
