# Структуры

Структуры (structs) - это коллекция полей:

```
type Vertex struct {
	X int
	Y int
}
```

Доступ к полям осуществляется через точку:

```
v := Vertex{1,2}
v.X = 4
```

Структуры доступны через указатель:

```
v := Vertex{1, 2}
p := &v
p.X = 1e9 // По хорошему мы должны были написать `(*p).X`, но Go для упрощения записи допускает писать просто `p` без разыменования.
```

Структурный литерал позволяет обозначить значение структуры путем перечисления его полей:

```
var (
	v1 = Vertex{1, 2}  // has type Vertex
	v2 = Vertex{X: 1}  // Y:0 is implicit
	v3 = Vertex{}      // X:0 and Y:0
	p  = &Vertex{1, 2} // has type *Vertex
)
```

Можно указать только часть полей используя синтаксис `Name: `, при этом порядок указания полей не важен.

Префикс `&` возвращает указатель на значение структуры.
