# Срезы

Слайсы в отличие от массивов, имеют динамический размер.

Срез не хранит данные сам, а только указывает на диапазон элементов внутри массива.

Срез может смотреть на любую часть массива - конкретный элемент, интервал или весь массив, а также наращивать и уменьшать его длину не создавая новый массив.

Тип `[]T` это слайс элементов типа `T`.

Срез формируется из двух ограничителей - верхнего и нижнего разделенных `:`:

```
a[low:high]
```

Срез включает себя элемент с индексом low, но не включает в себя элемент с индексом high.

Срезы не хранят никаких данных, они только описывают секция указанного массива.

Изменения элементов среза также изменяет элементы указанного массива (на который ссылается срез). Другие срезы, ссылающиеся на тот же массив также будут видеть изменения.
