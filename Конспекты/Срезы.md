# Срезы

Слайсы в отличие от массивов, имеют динамический размер.

Срез не хранит данные сам, а только указывает на диапазон элементов внутри массива.

Срез может смотреть на любую часть массива - конкретный элемент, интервал или весь массив.

Тип `[]T` это слайс элементов типа `T`.

Срез формируется из двух ограничителей - верхнего и нижнего разделенных `:`:

```
a[low:high]
```

Срез включает себя элемент с индексом low, но не включает в себя элемент с индексом high.

Срезы не хранят никаких данных, они только описывают секцию указанного массива.

Изменения элементов среза также изменяет элементы указанного массива (на который ссылается срез). Другие срезы, ссылающиеся на тот же массив также будут видеть изменения.

Ссылка на массив в слайсе указывает на первый элемент массива на который ссылается. А последний элемент определяется длиной `len`. А `cap` - емкость, то сколько влезет элементов в слайс.

Если заранее известен размер будущего слайса, то стоит заранее аллоцировать память под него с помошью функции `make` чтобы избежэать создания новых массивов и копирований данных из массива в массив при его заполнении.

Слайсы можно объявить с помощью литералом подобно массивам, но без указания длины:
`[]bool{true, true, false}`

При указании слайса можно опускать стартовый и конечный элемент массива. Следующие записи будут эквивалентны:
```
a[0:10]
a[:10]
a[0:]
a[:]
```

Длина слайса может быть получена с помощью `len(s)` и отражает количество элементов в слайсе.
Емкость слайса может быть получена с помощью `cap(s)` и отражает количество элементов в массиве на который ссылается слайс начиная с первого элемента в слайсе.

Нулевое значение слайса - `nil`. Такой слайс имеет нулевую длину и емкость и не имеет ссылки на массив.

Срезы можно создавать с помощью встроенной функции `make`:
```
b := make([]int, 0, 5) // len(b)=0, cap(b)=5
b = b[:cap(b)] // len(b)=5, cap(b)=5
b = b[1:]      // len(b)=4, cap(b)=4
```

Указание емкости является опциональным.
